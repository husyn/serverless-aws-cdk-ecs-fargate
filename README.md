# serverless-aws-cdk-iac
Infrastructure As Code (IaC) using AWS CDK with Typescript to create a complete serverless application

## Services to create

- Subnets (3 subnets)
  - NACL
  - Security Groups

- Application Load Balancer
- ECS Cluster
  - ECS Service
  - Fargate task
- ECR
  - ECR LifeCycle Rules
